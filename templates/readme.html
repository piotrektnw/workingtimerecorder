Working Time Recorder - a simple app to track employees working-time.
  1. Key functionalities:
    • track employees working-time
    • register user
    • change password
    • show content depending on user status
  2. Working Time Recorder has been created using:
    • Flask - a Python framework
    • Databases with SQLite 3 - storing user information as well as time data
    • HTML, CSS, Bootsrap (v. 5.0) and JavaScript - visualization of content
Working Time Recorder is divided into two parts - admin and user.
  1. Admin panel contains following functions:
    • display records of all employees
    • filter table and display needed information
    • remove user
    • clear database (working time database)
  2. If you are interested in testing admin panel use following login data:
    • Login: admin
    • Password: admin
Working Time Recorder is divided into two parts - admin and user.
  1. User panel contains following functions:
    • record working time using 'start' and 'stop' buttons
    • take a break function (extends working time)
    • display user records history

Feel free to register using Register button and test 'User panel' functionalities.
